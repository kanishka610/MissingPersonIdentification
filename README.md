Ours is a missing person identification system where we find the missing person in a group image obtained through CCTV Survellience Video 
Main execution file is 'Full.py'
Our methodology follows prompting the user to choose among our two workflows
    a. when you have person identity photo and group photo alone
    b. when we know about dress or any material the missing person posses
Noise.py handles noise in an image
Recog.py helps us convert video to image frames and capture faces in the image
